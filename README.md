# 📅 Telegram Calendar Bot

Бот для уведомлений о встречах из Google Calendar. Автоматически отслеживает предстоящие события и присылает напоминания в Telegram.

## ✨ Функциональность

- 🔐 **Авторизация через Google OAuth 2.0** (с поддержкой PKCE)
- ⏰ **Настраиваемые напоминания** о встречах (от 1 до 1440 минут)
- 📅 **Просмотр истории** последних 10 завершённых встреч
- 📋 **Выбор конкретного календаря** (если их несколько)
- 👥 **Поддержка нескольких пользователей**
- 📊 **Логирование ошибок и событий**

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу, авторизация |
| `/set_reminder <минуты>` | Установить время напоминания (1-1440 мин) |
| `/list_calendars` | Показать список доступных календарей |
| `/select_calendar` | Выбрать календарь для уведомлений |
| `/history` | Показать историю последних 10 встреч |
| `/help` | Показать справку |
